# Templates

This directory contains templates for common development workflows.

## Available Templates

- **PR-REVIEW.md** - Structured PR review workflow
- **ISSUE-ANALYSIS.md** - GitHub issue analysis (bugs vs features)
- **STATUS-SUMMARY.md** - Comprehensive status update format
- **VISUAL-DEBUG-SCRIPT.md** - TUI rendering debug script template
- **KEY-TESTER-INSTRUCTIONS.md** - Keyboard input debugging instructions
- **RELEASE-TWEET.md** - Release announcement composition guide

## Usage

Reference these templates when Claude needs structured guidance for specific tasks.

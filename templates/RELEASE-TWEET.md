# Release Tweet Composition Template

Format:

```
pi <version> released. <one-liner summary>

- <change description, mention @contributor if external>
- <change description>
...

npm install -g @mariozechner/pi-coding-agent

Changelog:
https://github.com/badlogic/pi-mono/blob/main/packages/coding-agent/CHANGELOG.md
```

Rules:
- For external contributors, find their Twitter handle from GitHub profile
- Use casual tone
- Group related changes

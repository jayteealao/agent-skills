> **Additional context needed**: brand guidelines, current design direction.

Transform a design that is too timid, generic, or lacking presence into something with genuine visual strength and personality.

---

## Register

Brand: maximalism is available — full-drenched color, editorial typography, dominant visual weight, theatrical composition. One committed visual choice executed with conviction beats five cautious ones. Push the aesthetic lane named in the brief aggressively.

Product: one accent at a time. Bolder in product means clearer hierarchy, stronger typographic contrast, more decisive whitespace — not more color or decoration. The goal is clarity, not drama.

---

## Diagnose the timidity

Before prescribing changes, identify the source:
- **Typographic: flat scale** — headings too close in size to body, no weight contrast
- **Color: excessive restraint** — everything in the same value range, no accent with conviction
- **Spatial: uniform density** — everything the same spacing, no emphasis through whitespace
- **Voice: hedging** — design is apologizing for itself (small type, pale colors, abundant padding everywhere)
- **Compositional: no hierarchy** — all elements treated as equal importance

## Strengthen Typography First

Typography is the single highest-leverage change for making a design bolder.

- Increase the heading-to-body size ratio. If h1 is 2× the body, try 3–4×
- Add weight contrast: pair a heavy heading weight (700–900) with a light body (300–400)
- Tighten heading tracking (−0.02 to −0.05 em for display sizes)
- Increase line height in body text: 1.6–1.75 for comfortable reading
- Use font variation axes (optical size, weight) if the font supports it
- Check the font reflex-reject list (brand register) — if using Fraunces or Plus Jakarta Sans, consider replacing with something less common

## Commit to the color strategy

Identify the current color strategy and upgrade it one tier if it's underserving the design:

- Restrained → Committed: introduce one accent color at 30–40% of colored elements
- Committed → Full palette or Drenched: add secondary supporting color or let the dominant color own more surface area

In both cases: use OKLCH, reduce chroma as lightness approaches extremes.

## Add spatial drama

Bold designs use whitespace as emphasis, not as default padding:
- Tighten the spacing inside dense areas (related elements, lists, form rows)
- Expand the breathing room around featured elements (hero, CTAs, key content blocks)
- This contrast — dense vs spacious — creates emphasis without adding more elements

## Make one dominant choice

A bolder design usually has one committed large-scale choice:
- A color that owns a significant surface area
- A type size that demands attention
- An asymmetric layout or grid that has a clear POV
- A texture, pattern, or image treatment that creates depth

Pick one. Execute it fully. Don't add three half-committed moves.

## Validate

After changes:
- Does the design have a point of view?
- Would you recognize it as distinct from a generic AI-generated design?
- Is the visual hierarchy immediately obvious?
- Can you name the aesthetic lane?

## Never
- Multiple large-scale dominant choices competing for attention
- Bold for bold's sake — every increase in visual weight should serve hierarchy
- Decorating rather than composing (adding elements vs. making structural changes)
- Aggressive typography with poor color contrast (bold weight does not substitute for contrast ratio)

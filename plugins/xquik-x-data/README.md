# xquik-x-data

Claude Code plugin for planning Xquik REST API and remote MCP workflows from public source truth.

## Source Truth

- Docs: https://docs.xquik.com
- OpenAPI: https://xquik.com/openapi.json
- MCP manifest: https://xquik.com/.well-known/mcp.json
- MCP endpoint: https://xquik.com/mcp
- Source repo: https://github.com/Xquik-dev/x-twitter-scraper

## Skill

Use `/xquik-x-data:xquik-x-data` when planning Xquik-backed X data research, monitoring, exports, webhooks, or authenticated automation. The skill keeps authentication opt-in and tells the agent to verify public docs before naming endpoints, schemas, limits, or response fields.

# Agent Behavior Patterns Plugin

Common workflow skills extracted from agent usage patterns.

## Skills Included

### Sound Notification
Play audio when tasks complete. Useful for long-running builds or deployments.

**Triggers**: "play a sound when done", "notify me when finished"

### TUI Testing
Interactive testing workflow for TUI changes using tmux and debug logging.

**Triggers**: "test TUI", "debug TUI rendering"

### Design Document Creation
Create design documents and corresponding GitHub issues for new features.

**Triggers**: "create design doc", "write design document"

### Changelog Audit
Pre-release verification of changelog entries for all external contributor PRs.

**Triggers**: "audit changelog", "verify changelog before release"
